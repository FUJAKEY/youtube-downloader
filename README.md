# Красивый Сайт на Node.js

Это многоуровневый веб-сайт с современной архитектурой, построенный на Node.js и Express.

## Особенности

- **Архитектура**: Четкое разделение на маршруты, представления и контроллеры (MVC).
- **Дизайн**: Современный темный режим с градиентами и отзывчивым интерфейсом.
- **Технологии**: Express, EJS, CSS3.
- **CI/CD**: Настроен GitHub Workflow (версия 4) для автоматической проверки кода.

## Структура проекта

```
.
├── src/
│   ├── app.js          # Точка входа
│   └── routes/         # Маршрутизация
├── views/              # EJS Шаблоны
│   ├── partials/       # Части (шапка, подвал)
│   └── ...             # Страницы
├── public/             # Статические файлы (CSS)
└── .github/workflows/  # GitHub Actions
```

## Как запустить локально

1. Установите зависимости:
   ```bash
   npm install
   ```

2. Запустите сервер:
   ```bash
   npm start
   ```

3. Откройте в браузере: `http://localhost:3000`

## GitHub Workflow

В репозитории настроен файл `.github/workflows/main.yml`, который автоматически запускает проверку проекта при каждом пуше в репозиторий. Это гарантирует, что сайт всегда запускается корректно.

Для развертывания сайта в интернет (чтобы вы могли зайти на него с любого устройства), рекомендуется использовать такие сервисы как Render, Railway или Heroku, подключив их к вашему GitHub репозиторию.
